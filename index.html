操作說明

簡介

本工具將 120+ 筆真實泌尿科急診照會整理為 19 個診斷分類、41 條處置路徑，並收斂出 61 條可複製的英文診斷骨架，幫助你快速產生英文照會回覆。

步驟一：選擇診斷分類

在搜尋欄輸入關鍵字（如 "hematuria"、"stone"、"torsion"），或點擊下方快捷按鈕。左欄 (Diagnosis) 會篩選出對應的診斷分類，點選一個分類。

步驟二：複製具體診斷 / 選擇處置

點選分類後，畫面往右展開：

  • 第 2 欄 (Specific Dx)：列出該分類的具體英文診斷骨架，點任一條即複製到剪貼簿，貼進病歷 impression。側別 (left/right)、大小 (cm)、日期請自行補上。
  • 第 3 欄 (Management)：列出該分類的處置路徑（如 Surgery、Conservative、MET + OPD），點選後右欄出現範本。

步驟三：複製處置範本

點選處置路徑後，最右欄顯示英文照會範本。你可以：

  • 直接編輯文字內容
  • 點 Copy All 一鍵複製
  • 點 Reset 回復原始範本

特殊功能：TWIST Score 計算器

選擇「Testicular Torsion / Pain」→ 點選 Assessment 欄 → 點 TWIST Calc 按鈕，互動勾選計算分數。Assessment 區域可直接編輯，點 Copy 複製、Reset 回復原始範本。

鍵盤快捷鍵

  • / 聚焦搜尋欄
  • Esc 清空搜尋 / 關閉視窗
  • ↑ ↓ 切換診斷分類或處置路徑
  • ⌘/Ctrl + Enter 複製目前範本

離線使用 (PWA)

本工具支援 PWA — 第一次載入後即可離線使用。手機可在 Safari 點「分享 → 加入主畫面」、Chrome 可點「安裝應用程式」，加到主畫面後支援全螢幕、離線、開機快速啟動。

淺色 / 深色模式

右上 A / ☀ / ☾ 按鈕循環切換主題：A 跟隨系統 → ☀ 強制淺色 → ☾ 強制深色。設定會記在這個瀏覽器裡。

列印

瀏覽器 ⌘/Ctrl + P 可只列印目前 Template 內容（自動隱藏其他欄位）。

小提示

  • 可輸入多個關鍵字（如 "stone infection"），會以 AND 條件篩選；搜尋也會比對具體診斷文字
  • 所有診斷與範本皆源自真實照會紀錄，非 AI 生成內容
